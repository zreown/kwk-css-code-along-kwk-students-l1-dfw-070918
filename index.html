What is CSS?

CSS stands for Cascading Style Sheets. We use CSS to apply style and formatting to websites. It's a best practice as a programmer to keep content and styling separate. That way we can keep our files simple and also allow for different webpages to have the same styling! Imagine if every time the New York Times had to publish an article they had to re-do the look of the entire newspaper!

  • Backgrounds

  • Page Layout
  • Margins
  • Image sizes
  • Rounding edges
  • LOTS MORE...

Tell me about CSS Syntax...

CSS syntax is pretty straightforward. The selector chooses the element whose formatting you want to change. The declaration explains how to change the element

How do we get CSS to affect HTML

You need to create a CSS stylesheet and link to it in your HTML file. Your link should be in the 'head' section of the html and should look something like this:
<link rel="stylesheet" type="text/css" href="path_to_stylesheet">
Do this now.

What are these things we call classes and ids?

Sometimes we have to get more granular than what the tags in our html can provide us. For example, we only want to convert one of our h3 headings to a different color. If we change the css of h3s, the will all change. So we add classes and/or ids. Classes and ids are 'markers' that we put in our HTML so that we can reference them in our CSS stylesheets. An id can only be used once in a page, but a class can be used multiple times. In our stylesheets, ids are referenced with a '#' and classes are referenced with a '.'.
