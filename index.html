An Introduction to Cascading Style Sheets

Welcome! We're excited to have you here. Today, we'll be walking through learning CSS. It's going to be awesome.

What is CSS again?

CSS stands for Cascading Style Sheets. We use CSS to apply style and formatting to websites. It is a good practice as a programmer to keep content and styling separate. It helps keep our code organized, easy to read, and reuseble. CSS written for one page can be linked by multiple pages. Different pages on a website can share a local file, often stored in a CSS subfolder. Different websites can even share CSS that is hosted remotely and accessed when a page loads.

Tell me about CSS Syntax...

CSS syntax is very consistent, regardless of what you're styling. The selector chooses the element whose formatting you want to change, in this example, all h1 tags. The declaration explains how to change selected the element. Here, all h1 tags will have their text changed to blue and the font size set at 12 pixels.

Make sure to add those semi-colons to the end of each declaration! You can have as many declarations as you like for each selector.

It is possible to not only turn all h1 elements blue, but also make them to different fonts and sizes, give them borders, turn them upside-down, add them 3d effects, and even animate them!

How do we get get started on combining CSS HTML?

The first thing you need to do is create a CSS stylesheet and link to it in your HTML file. Your link should be in the 'head' section of the html in this page and should look something like this:

<link rel="stylesheet" type="text/css" href="path_to_stylesheet">

There is a provided style.css file inside of the css folder in this code along. Go ahead and add a link tag to index.html in the 'head' section. The href should be set to "css/style.css"

Write your first CSS Declarations

Now that style.css is linked to this webpage, we can start seeing some results when we style. Go into style.css, and add the following:

    body {
      background-color: lightblue;
      font-size: 150%;
    }

Save the file and refresh this page. Whoa! The white background should now be a nice color and all our text is bigger!

Curious! Everything got bigger!

Feel free to set font-sizes back to normal. We set a font-size using the body tag, but did you notice that EVERYTHING got bigger? All the text on this page is inside 'p' and 'div' tags.

Because the 'body' tag contains all the 'p' tags and all the 'div' tags, some values we set, like font-size, will trickle down to the elements inside

What are classes and ids?

Selecting by tag like we just did works well when we are making changes to every instance of a tag (all 'h1', all 'p' tags, etc..).

Sometimes, we want only some 'p' tags to look a certain way, or even one specific tag. In order to change specific elements, we use CSS classes and ids.

To implement a class, we will need to write a CSS block just like we did with 'body'. For example: 

    .myClass {
      background-color: gray;
      color: blue
    }

To write declarations for a class, first decide on a name. Writing '.', then the name will let CSS know these are class declarations

Now, just this one 'h3' element will have a gray background and blue text! Not only that, any element that is given the class "myClass" will have a gray background and blue text.

Let's practice this. First, create a class in 'style.css'. It can be called anything you like. Your name, your favorite color, etc..

Next, add some declarations. If you're not sure what to add, try some or all of the following, and feel free to change the values:

    .anExampleClass {
      font-size: 300%;
      background-color: lightgreen;
      font-family: courier;
      border: 1px dotted red;
      text-transform: uppercase;
      letter-spacing: 5px;
      transition: rotate(-30)deg;
    }

Finally, with our class declarations created, we need to assign this class to elements in index.html

Open index.html in your IDE, choose some HTML elements and add 'class=' followed by the name of your new class, in quotes. Maybe try adding a class to the `h1` and `h2` elements at the top of the page.

Save the work you've done, and refresh this page. The elements you've selected should display the styling you've declared.
